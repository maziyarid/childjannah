## [3.1.0] - 2026-02-23
### Added
- Inlined the Table of Contents, Polls, Star Rating, Key Takeaways, and FAQ Schema modules directly under `inc/`.
- Created `LICENSE`, `CHANGELOG.md`, and `CONTRIBUTING.md` to formalize project governance.
- Added a GitLab CI pipeline for PHP linting and downloadable theme archives.
- Published a GitLab Wiki home page summarizing setup, deployment, and feature modules.

### Removed
- Deleted the legacy `Snippets/` directory after migrating every referenced module into the theme.

### Changed
- Updated repository documentation to reflect the leaner structure with no external snippets.
