# Teznevisan3.com SEO & Technical Recovery Project

This repository contains the technical and SEO recovery tasks for **teznevisan3.com**. The purpose of this project is to clean up spam injections, fix technical SEO issues, enhance content optimisation, and improve website performance to recover search rankings and overall visibility.

## Project Goals:
- **Spam Cleanup**: Identify and remove spam URLs, files, and patterns that are affecting indexing and SEO performance.
- **Technical Fixes**: Resolve issues with redirects, indexing, blocked resources, and improve crawl efficiency.
- **On-Page Optimisation**: Add missing metadata (meta descriptions, titles, alt texts), implement SEO best practices, and optimise featured images.
- **Content Expansion & Optimisation**: Improve existing content, optimise keyword usage, and enhance user engagement signals.
- **Backlink Cleanup**: Disavow harmful or spammy backlinks and improve the website's link profile.

## Project Structure:

### Folders and Files:
- **/src**: Contains scripts and tools to implement SEO fixes, such as redirects, canonical tags, and content updates.
- **/docs**: Documentation for the recovery process, detailed explanations for SEO strategies, and configuration files.
- **/assets**: Contains assets like images, logos, and templates for featured images.

### Key Files:
- `htaccess_rewrite_rules.txt`: Contains `.htaccess` configuration for spam blocking and redirect fixes.
- `robots.txt`: The robots configuration file for controlling search engine crawlers.
- `SEO_Checklist.md`: Markdown file with tasks and progress tracking.
- `sitemap.xml`: Clean XML sitemap after spam URL removal.
- `disavow_file.txt`: List of backlinks to be disavowed to prevent spam influence.

## Setup Instructions:

1. **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/teznevisan3-seo-recovery.git
    cd teznevisan3-seo-recovery
    ```

2. **Update .htaccess**:
    - Open the `.htaccess` file and apply the changes provided in `htaccess_rewrite_rules.txt`.

3. **Update robots.txt**:
    - Replace your current `robots.txt` with the version provided in this repository to block spam crawlers and allow essential resources.

4. **SEO Task Checklist**:
    - Use the tasks listed in `SEO_Checklist.md` to track your progress.
    - Follow the detailed steps to implement technical fixes, content optimisations, and performance improvements.
